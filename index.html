CALCULATOR MEDIA CU TEZA DESCRIERE

CUM SE CALCULEAZA MEDIA CU TEZA

Calcularea mediei cu teza se face in doua etape in felul urmator:

In prima etapa se calculeaza media aritmetica a tuturor notelor obtinute la o anumita materie de-a lungul unui semestru. Media se face la doua zecimale exacte fara rotunjire.

In a doua etapa se se aplica formula MF = (3M + T)/4. MF este media finala pentru materia cu teza, M este media notelor obtinute in prima etapa, aceasta seinmulteste cu 3 se aduna cu teza dupa care se imparte la 4.

Calcularea mediei cu teza se face insa foarte usor cu calculatorul de mai sus. Descrierea aceasta este informativa doar ca sa stiti cum se calculeaza media cu teza.

Media notelor se face la doua zecimale exacte fara rotunjire. De exemplu o medie de 8.457 nu se rotunjeste la 8.46 ci se considera 8.45. Aceasta medie (8.45) o inmultim cu 3 iar rezultatul (25.35) se aduna cu nota din teza, sa zicem (9) => 25.35 + 9 = 34.35, dupa care se imparte la 4 => (8.58). Rezultatul calcului se rotunjeste la o nota intreaga, in cazul nostru 9.

CALCULATOR MEDIE CU TEZA EXEMPLE

Mai jos aveti 3 exemple cum se calculeaza media cu teza. Calcul detaliat pas cu pas al mediei cu teza.

PASUL 1:

SE CALCULEAZA MEDIA NOTELOR - M
  • Ex 1: M = (10 + 10 + 9)/ 3 = 9.66 ( 2 zecimale exacte fara rotunjire)
  • Ex 2: M = (9 + 7 + 9 + 10)/ 4 = 8.75 ( 2 zecimale exacte fara rotunjire)
  • Ex 3: M = (7 + 10 + 8 + 8 +9)/5 = 8.4 ( 2 zecimale exacte fara rotunjire)

Se calculeaza media aritmetica a tuturor notelor obtinute de-a lungul unui semestru la o anumita materie. Adunam toate notele si le impartim la cate note sunt, facem astfel media aritmetica.

PASUL2 :

CALCULAREA MEDIEI FINALE PENTRU MATERIILE CU TEZA
  • Ex 1: (3M + T)/4 = (3 * 9.66 + 9)/4 = 9.495 → 9
  • Ex 2: (3M + T)/4 = (3 * 8.75 + 9)/4 = 8.8125 → 9
  • Ex 3: (3M + T)/4 = (3 * 8.40 + 9)/4 = 8.55 → 9

Media notelor plus teza impartit la patru - (3M + T)/4. Rezultatul mediei aritmetice de la pasul 1 (cu 2 zecimale exacte fara rotunjire) il inmultim cu 3, adunam cu teza iar suma acestora o impartim la 4. MF = (3M + T)/4
